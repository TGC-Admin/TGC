問題文
=====
$(1, 2, \ldots, N)$ の順列 $P(P_1, P_2, \ldots, P_N)$ があります．  
また，次の操作を $0$ 回以上好きなだけ行うことができます：  
- $1 \leq i, j \leq N$ を満たす整数の組 $(i, j)$ を選び，$P_i$ と $P_j$ の値を入れ替える．  

$P$ を昇順に並び替えるために必要な操作回数の最小値を求めてください．  

制約
-----
- $1 \leq N \leq 10^5$
- $P$ は $(1, 2, \ldots, N)$ の順列である

入力
-----
入力は以下の形式で標準入力から与えられる．
```md
$N$ 
$P$
```

出力
-----
答えを出力せよ．
