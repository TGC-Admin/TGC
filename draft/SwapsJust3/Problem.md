問題文
=====
`A`, `B`, `C` のみからなる，長さが $3$ の文字列 $S$ に対して以下の操作を行うことができます：
- $1$ 文字目と $2$ 文字目を入れ替える
- $2$ 文字目と $3$ 文字目を入れ替える

操作をちょうど $K$ 回行って，文字列 $S$ を `ABC` に一致させることはできますか？  
判定してください．  

制約
-----
- $0 \leq K \leq 10^{18}$

入力
-----
入力は以下の形式で標準入力から与えられる．
```md
$S$  
$K$
```

出力
-----
答えを出力せよ．  

サンプル
=====
```入力例1
CBA
4
```
```出力例1
Yes
```
`CBA` $\to$ `CAB` $\to$ `ACB` $\to$ `ABC` と変化させることができます．  

```入力例2
CAB
1
```
```出力例2
No
```


```入力例3
ABC
0
```
```出力例3
Yes
```
